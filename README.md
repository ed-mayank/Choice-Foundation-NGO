## Starting

Install the required dependencies for both frontend and backend

```bash
❯ cd src/backend
❯ yarn install
```

```bash
❯ cd src/frontend
❯ yarn install
```

Start backend: `❯ yarn run dev`

Start frontend: `❯ yarn run start`

ENJOY!

NOTE: Backend runs on PORT 3001 and Frontend on PORT 3000.
